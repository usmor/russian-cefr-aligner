Эта папка репозитория содержит финальный код с использованием библиотеки trl.

В файле ```trl.ipynb``` происходит загрузка модели-симплификатора, загрузка берта для оценки уровня текста, загрузка функции вознаграждения. Затем приведён необходимый код для реализации 
обучения с подкреплением с помощью библиотеки trl (transformer reinforcement learning). В конце файла оценивается работа двух моделей симплификаторов qwen и meno-tiny на тестовом датасете.

Qwen чуть лучше справляется с генерацией упрощённых предложений и с точки зрения accuracy (0.18 > 0.166), и с точки зрения осмысленности получаемых текстов. Низкое значение accuracy 
может обуславливаться неидеальностью работы берта. Например, предложениям "Ничего не получилось" / "Лиза побегала по городским улицам" он присваивает уровень C2, хотя, очевидно, это не так.

**Пример работы квена**:

ОРИГИНАЛ:

- *Они хотели, чтобы всё оставалось чётким и холодным, без чувств и переживаний.*

УПРОЩЁННЫЙ ТЕКСТ (УРОВЕНЬ: A1, ЦЕЛЬ: A1):

- *Они хотели, чтобы мир был спокоен и неподвижен, как камень.*

**Пример работы мено**:

ОРИГИНАЛ:

- *Князь немедленно хотел поворотить назад к себе, в гостиницу; даже повернулся и потел; но чрез минуту остановился, обдумал и воротился опять по прежней дороге.*

УПРОЩЁННЫЙ ТЕКСТ (УРОВЕНЬ: B2, ЦЕЛЬ: B2):

- *Князь хотел повернуть назад к себе, в гостиницу, но через минуту остановился, обдумал и воротился опять по прежней дороге.*

В файлах формата ```.csv``` содержатся соответствующие csv-файлы со сгенерированными симплификациями для двух из моделей, а также с целевыми и предсказанными уровнями.
