В данной папке находится код с дообучением различных конфигураций BERT на задачу классификации русских текстов по шкале CEFR (A1-C2). 
Лучший перформанс у более современной модели Ai-forever-ruBert-base -- accuracy 73.36, F1-score 73.5. 
Zip-файл с дообученной моделью лежит по этой ссылке: https://disk.360.yandex.ru/d/WgOiW6udSSlcOg

Помимо дообучения классификатора, в файле project_nlp1.ipynb так же находится reward function, 
её результаты с различными параметрами на датасете с парами текст-упрощенный текст для выбора наилучших аргументов находятся в первом скриншоте
